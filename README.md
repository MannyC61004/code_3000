# code_3000
Collection of Assignments for CSE 3000: Contemporary Issues in Computer Science and Engineering

Personal Statement on Ethics:

Accountability must be built into the code and data from the start, rather than corrected after a system is deployed.

As systems become more complex, maintaining clear and explainable logic is vital to prevent bias.

Technical efficiency should never come before human safety or equity. The  goal of any machine should be to support and protect human values.
